# Emotion thermometer — English documents

These files are **full English translations** of the Dutch materials in `examples/`.

| File | Source | Purpose |
|------|--------|---------|
| [Your-Thermometer.md](./Your-Thermometer.md) | `Jouw_thermometer.pdf` | **Blank template** — fill in your own rows |
| [Example-Angry-Thermometer.md](./Example-Angry-Thermometer.md) | `Voorbeeld thermometer boos.docx` | Completed example (anger) |
| [Example-Scared-Thermometer.md](./Example-Scared-Thermometer.md) | `Voorbeeld Thermometer.docx` | Completed example (fear) |

## How to use

1. Open **Your-Thermometer.md** and replace the empty cells with your own words (in any editor, or on the website in `site/`).
2. Read the examples for inspiration; copy rows you like into your own thermometer.
3. Print or save as PDF from your browser if you use the web pages (`site/`).
